Inside a daily exam:

Floating Zed button is hidden. Bottom nav is already gone. Only the question, the timer, and the section dots remain — no nudges, no chat, no leaderboard popups.

What changes for the learner

  • → Decision removed. The dashboard tells them what to do next instead of asking them to choose.
  • → Less visual noise. Floating stars, four big gradient buttons, and bouncing emojis are demoted to a single chip row.
  • → Weakness-driven. Step 1 is always the worst topic — the same data we already compute for the premium "Weak Topics" card.
  • → Distraction-free runner. Zed FAB hidden on /exam/:id. Re-enabled on results.
